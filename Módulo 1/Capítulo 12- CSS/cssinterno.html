CSS Interno/Local

Usando as CSS localmente, estilo e conteúdo ficam separados no código, o que facilita o trabalho.

Para usá-las, use a tag <style> entre as tags de <head>

A partir daí, use os seletores para estilizar os elementos.

As desvantagens são que o códgio fica muito grande, e se for mais de uma página, será necessário colocar a tag <style> em todos os arquivos HTML.
